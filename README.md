# Java ArrayList Practice

This repository contains multiple examples of using `ArrayList` in Java.

## Files

- `Main.java` – basic ArrayList with Strings
- `Foods.java` – ArrayList with favorite foods
- `PArrayList.java` – ArrayList with different loop styles
- `numberArrayList.java` – ArrayList of numbers with sum and loops
- `ArrayList.java` – general example of ArrayList

## Topics Covered

- Creating an `ArrayList`
- Adding elements (`add()`)
- Accessing elements (`get()`)
- Getting size (`size()`)
- Loops: `for`, `for-each`
-remove elements (remove() )
- Clear elements ( clear() )
- Calculating sum

 How to Run

Compile and run any `.java` file individually.
